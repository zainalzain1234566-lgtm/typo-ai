# Typo AI — وصف المشروع

> أداة عربية لتوليد كاروسيلات إنستغرام بالذكاء الاصطناعي، من فكرة نصية إلى شرائح مصمّمة جاهزة للنشر.

## نبذة

Typo AI منصّة ويب تساعد صانعي المحتوى الناطقين بالعربية على إنشاء كاروسيلات (منشورات متعددة الشرائح) لإنستغرام بكفاءة. يكتب المستخدم موضوعه، يحدّد تخصيصات المحتوى (النوع، الجمهور، الأسلوب، اللهجة، عدد الشرائح)، يختار قالبًا تصميميًا، فيتولّى التطبيق: كتابة النصوص عبر نموذج ذكاء اصطناعي، تطبيق التصميم، ثم تصدير الصور بدقة عالية.

## المشكلة التي يحلّها

إنشاء كاروسيل إنستغرام احترافي يتطلب خبرة في الكتابة والتصميم وأدوات معقدة (Photoshop/Canva)، مع دعم محدود للغة العربية واتجاه RTL. Typo AI يدمج الكتابة والتصميم في خطوة واحدة: فكرة → شرائح مكتوبة ومصمّمة → صور PNG جاهزة.

## المميزات

### توليد محتوى بالذكاء الاصطناعي
- نموذج DeepSeek عبر OpenRouter يكتب الشرائح كاملة (عناوين، نصوص، دعوات إجراء)
- دعم 8 أنواع محتوى: تعليمي، قصة، توعوي، قائمة، خطوات، نصائح، مقارنة، شرح مفهوم
- دعم 5 لهجات/لغات: العربية الفصحى، العراقية، الخليجية، المصرية، الإنجليزية
- تخصيص النبرة (8 خيارات)، المستوى (مبتدئ/متوسط/متقدم)، الجمهور المستهدف
- توليد تلقائي للـ Caption وHashtags جنبًا إلى جنب مع الشرائح

### 20 قالبًا تصميميًا
- **10 قوالب أصلية**: تحرير، واضح، نقطة، إطار، موجز، أكاديمي، حديث، تبایين، شبكة، عنوان
- **10 قوالب جديدة**: Hero، Editorial، Split، Stacked، Cards، Rotated، Terminal، Magazine، Tilt، Retro
- كل قالب يدعم 4 لوحات ألوان + خطوط متنوعة (Tajawal، Cairo، IBM Plex Sans Arabic)
- تبديل القوالب دون فقدان المحتوى

### محرر شرائح
- تعديل النصوص بعد التوليد
- إعادة ترتيب الشرائح بالسحب
- إضافة/حذف شرائح فردية
- مقياس حجم الخط قابل للتعديل (0.8x – 1.3x)

### الهوية البصرية (Brand Kit)
- حفظ الشعار، اسم الحساب، اللون الأساسي، الخط الافتراضي
- خيارات العرض: إظهار/إخفاء الشعار، اسم الحساب، رقم الشريحة
- تحديد موضع العناصر (أربع زوايا)

### التصدير
- تنزيل شريحة واحدة كصورة PNG
- تنزيل جميع الشرائح في ملف ZIP
- دقة 1080px بدون علامة مائية
- ثلاثة مقاسات: مربع (1:1)، عمودي (4:5)، ستوري (9:16)

### إدارة المشاريع
- لوحة تحكم بإحصائيات (مشاريع، تصديرات، قوالب مفضلة)
- تنظيم المشاريع في مجلدات
- تمييز المشاريع المفضلة
- تكرار المشاريع

## التقنيات

### الواجهة الأمامية (Frontend)
- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** للتنسيق
- **Framer Motion** للحركات
- **React Hook Form + Zod** للنماذج والتحقق
- دعم كامل لـ RTL والعربية
- خطوط Google: Tajawal، Cairo، IBM Plex Sans Arabic، Playfair Display، Space Grotesk، IBM Plex Mono، Courier Prime

### الخادم (Backend)
- **Supabase** (PostgreSQL + Auth + Storage + RLS)
- جلسات SSR قائمة على الكوكيز عبر `@supabase/ssr`
- Server Actions للعمليات (إنشاء/تحديث/حذف المشاريع والشرائح)
- استعلامات العميل تحت RLS بمفتاح publishable
- مفتاح service-role في السياقات الخادمية فقط
- 10 ترحيلات SQL (جداول، تريجرز، RLS، تخزين، RPCs، فهارس، بيانات أولية)

### الذكاء الاصطناعي
- **OpenRouter** كبوابة وصول للنماذج
- نموذج **DeepSeek** افتراضيًا
- Prompt نظامي عربي بقواعد لهجات صارمة
- تعليمات خاصة بكل نوع محتوى
- استخراج JSON من استجابات قد تحتوي رموز markdown

### التصدير
- **html-to-image** لتحويل مكونات React إلى PNG
- **JSZip** لحزم الشرائح في أرشيف

## بنية قاعدة البيانات

| الجدول | الوصف |
|--------|-------|
| `profiles` | ملفات المستخدمين مرتبطة بـ auth.users |
| `brand_kits` | الهوية البصرية لكل مستخدم |
| `user_preferences` | الإعدادات الافتراضية لكل مستخدم |
| `folders` | مجلدات تنظيم المشاريع |
| `templates` | القوالب التصميمية (slug، مفتاح المكوّن، الترتيب) |
| `template_palettes` | لوحات ألوان كل قالب (4 لكل قالب) |
| `favorite_templates` | القوالب المفضلة للمستخدم |
| `projects` | المشاريع (الموضوع، التخصيصات، الإعدادات) |
| `slides` | شرائح المشاريع (الموضع، النوع، النصوص) |
| `export_records` | سجل عمليات التصدير |
| `generation_jobs` | سجل مهام توليد الذكاء الاصطناعي |
| `account_usage` | عدّاد الاستخدام لكل مستخدم |

## تدفق العمل

```
مستخدم يكتب الموضوع
        ↓
يختار التخصيصات (نوع، جمهور، أسلوب، لهجة، عدد شرائح)
        ↓
Server Action → ExternalAIProvider → OpenRouter/DeepSeek
        ↓
JSON (slides + caption + hashtags)
        ↓
حفظ في projects + slides (PostgreSQL)
        ↓
محرر الشرائح (تعديل، إعادة ترتيب، تغيير قالب/ألوان)
        ↓
تصدير PNG مفرد أو ZIP (html-to-image)
```
